🦉 PMP Quest

Install as a home screen app on Android — no app store needed.

Option A — Use GitHub Pages (recommended, free hosting)

1
Create a free GitHub account
Go to github.com and sign up if you don't have one.
2
Create a new repository
Click + → New repository. Name it pmp-quest. Set it to Public. Click "Create repository".
3
Upload the files
Click "uploading an existing file". Drag and drop all 4 files:
index.html   manifest.json   sw.js   and the icons/ folder contents.
Commit changes.
4
Enable GitHub Pages
Go to your repo → Settings → Pages → Source: main branch → Save.
Your URL will be: yourusername.github.io/pmp-quest
5
Install on Android
Open the URL in Chrome on your Android phone. Tap the 3-dot menu (⋮) → "Add to Home screen" → Add. Done!
⚡ Once installed, the app works fully offline. Your progress (XP, streaks, spaced repetition data) is saved locally on your phone.

Option B — Local network (no internet hosting)

1
Install Node.js on your computer
Download from nodejs.org (LTS version).
2
Serve the files
Open Terminal in the pmp-pwa folder and run:
npx serve . --listen 3000
3
Open on Android
Make sure phone and computer are on same WiFi. Find your computer's local IP (e.g. 192.168.1.5) and open http://192.168.1.5:3000 in Chrome on your phone. Add to home screen.

What you get

✓ Looks and feels like a native Android app
✓ Works completely offline after first load
✓ Spaced repetition progress saved locally
✓ 45-question prep exam with 60-min timer
✓ Domain weakness radar chart after every session
✓ No app store, no account, no cost
